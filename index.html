A quoi sert ce site ?

Ce site sert a s'entrainer a trouver rapidement le jour de la semaine d'une date avec la methode Doomsday.

Calcul de K, a, b, c et D

Pour une annee Y :

C = floor(Y / 100) (siecle)

y = Y mod 100 (deux derniers chiffres)

K = (5 * (C mod 4) + 2) mod 7

a = floor(y / 12)

b = y mod 12

c = floor(b / 4)

D = (a + b + c + K) mod 7

D est le jour doomsday de l'annee (0 = dim, 1 = lun, ...).

14/3, 28/2 (ou 29/2 en annee bissextile)

Janvier : 3/1 (ou 4/1 en annee bissextile)
